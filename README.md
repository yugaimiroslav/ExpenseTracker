https://roadmap.sh/projects/expense-tracker

## How to Run

### Prerequisites
Make sure you have [Go](https://go.dev) installed on your machine.

### Building the Project
To compile the project and create an executable file, run:
```bash
go build -o expense-tracker
```
This will generate an executable named `expense-tracker` (or `expense-tracker.exe` on Windows).

### Running the Application
You can run the project directly without building:
```bash
go run main.go
```
Or, if you have already built the executable, run:
```bash
./expense-tracker
```
